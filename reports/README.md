# Reports (storici)

Questa cartella contiene report storici generati durante review/refactor precedenti.
Non sono necessariamente allineati allo stato attuale del codice.

Per lo stato corrente del progetto fare riferimento a:
- `docs/PROJECT_STATUS.md`

Se un report storico deve diventare parte della documentazione ufficiale, va
validato e aggiornato per riflettere l architettura attuale.
